# 🖨️ نظام إدارة وقراءة البطاقة التموينية الإلكترونية العراقية

برنامج مكتبي ذكي مصمم لمساعدة أصحاب المكاتب ومراكز الطباعة على سحب وقراءة بيانات البطاقة التموينية بالاعتماد على الذكاء الاصطناعي (Gemini API)، لتجهيز الكروت للطباعة الفورية أو إرسالها إلى التلغرام بلمح البصر.

## ✨ الميزات الرئيسية
- **قراءة تلقائية (OCR):** يقرأ الاستمارات المستخرجة تلقائياً دون كتابة يدوية.
- **تعديل يدوي كامل:** إمكانية تعديل وتعديل الحقول بدقة قبل الطباعة.
- **جدول أفراد عائلة ديناميكي:** يتسع لعدد لا نهائي من الأفراد دون اقتطاع من الكرت المطبوع.
- **أمان عالي:** تشفير الحقول الحساسة (Token & ID) بنجوم حماية لمنع التجسس.
- **التكامل مع التلغرام:** إمكانية إرسال تقرير البطاقة التموينية بكامل تفاصيلها لجروب أو بوت تلغرام.
- **طباعة احترافية:** تصدير الكرت لصفحة ويب قياسية للطباعة الفورية بـ (Ctrl + P) مع الهيدر والشعارات يميناً ويساراً (`logo1.png` و `logo2.png`).
